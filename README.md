# markdown-editor

This is a cool Markdown Editor implemented in ClojureScript.

![alt screenshot](screencast.gif)

## Overview

The project uses ClojureScript, Reagent, and shadow-cljs.

## Develop

```
$ npm install
$ npm run watch
```

Then go to http://localhost:8080/ in your browser.

## Release

```
$ npm run release
```

## Features

- Edit markdown
- Download markdown
